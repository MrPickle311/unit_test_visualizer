Detailed Description

This class contains settings for a port.

This is package which can be passed to backend::PortStateOperator::changeSettings() to change current port settings. This class provides getters and setters for each setting. This class allows to set the following options:

  • baud rate
  • data bits
  • flow control
  • parity bits
  • stop bits
