Detailed Description

template<typename BufferType, typename PortOperatortType>
class bridge::Program< BufferType, PortOperatortType >

This template class allows to create program object.

Object of this class binds all other objects to Qt Meta-Object System , to expose their functionalites to QML. It also makes appropriate binding between objects. Object of this class requires to work pointer and template dependencies. This approach allows to easily change part of implementation in the future. This object requires 3 pointer dependencies which interfaces are following:

  • interface::PortScanner
  • interface::Converter
  • interface::ParserComponent

It also requires two template implementation dependencies :
