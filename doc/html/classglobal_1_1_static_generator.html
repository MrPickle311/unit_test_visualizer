Detailed Description

template<typename KeyType, typename ValueType>
class global::StaticGenerator< KeyType, ValueType >

This class template allows to create singleton which stores static collection of object.

Objects are mapped , so access to them is possible by a key. To create singleton user has to specialize initValues() method and put there certain objects. Objects are binded by key-value map ,so this template has two parameters:

Parameters
KeyTypeType of keys in the map
ValueTypeValue of keys in the map

This class is threadsafe
