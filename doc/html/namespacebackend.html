◆ StandardSettings

using backend::StandardSettings = typedef global::StaticGenerator<StandardSetting , PortFlowSettings>

This is generator which holds most frauqently used port settings. It stores these settings in PortFlowSettings objects. Each stored setting can be accesed by a key of StandardSetting enum type. Every standard settings have common options:

  • 8 data bits
  • no flow control
  • 1 stop bit
  • no parity

They differ only in baud rate
